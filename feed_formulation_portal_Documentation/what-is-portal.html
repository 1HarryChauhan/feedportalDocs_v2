What is the Feed Formulation Portal?

The Feed Formulation Portal is a web-based application designed to streamline and modernize feed formulation management for real-world use cases. Built as a capstone project for Deakin University, it provides an intuitive interface for users to manage ingredients, feeds, and species, and includes robust authentication, validation, and business logic suitable for production environments. It follows the MERN stack architecture (MySQL, Express.js, React.js, Node.js) and uses Clerk.js for secure authentication.

This portal is intended as a reference and foundation for future student cohorts, ensuring continuity and scalability of the project.
